## What I'm Currently Building

### Job Card System
A field operations platform for engineering teams. Eliminates 
the need for engineers to visit the office before every job — 
instead, they receive email notifications with full job details 
and go directly to site. Supervisors track job progress in 
real time (pending → in progress → completed) and can initiate 
M-Pesa STK Push payments directly from the dashboard.

**Stack:** Node.js · Express.js · PostgreSQL · React · Tailwind CSS  
**Integrations:** M-Pesa Daraja API · Paystack · Nodemailer
